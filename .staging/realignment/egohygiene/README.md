# egohygiene

This is the **schema-driven spiritual monorepo** for the [Ego Hygiene](https://egohygiene.io) ecosystem.

## Structure

- `apps/`: sites, dashboards, docs
- `packages/`: schema, themes, content, components
- `cli/`: scaffolding tools
- `tools/`: generators, automation
