# Awesome Obsidian

> Curated references for building and maintaining the `garden/` vault's
> appearance library.

## Core references

- [Awesome Obsidian](https://github.com/kmaasrud/awesome-obsidian) — source of
  curated community CSS snippets
- [Awesome Obsidian Plugins](https://github.com/noghartt/awesome-obsidian-plugins) —
  curated plugin and theme discovery list
- [Obsidian Help — Themes](https://github.com/obsidianmd/obsidian-help/blob/master/en/Extending%20Obsidian/Themes.md) —
  official theme installation and management guidance
- [Obsidian Help — CSS snippets](https://github.com/obsidianmd/obsidian-help/blob/master/en/Extending%20Obsidian/CSS%20snippets.md) —
  official snippet workflow and storage layout
- [Obsidian Help — Settings](https://github.com/obsidianmd/obsidian-help/blob/master/en/User%20interface/Settings.md) —
  confirms `.obsidian/themes/` and `.obsidian/snippets/` locations
- [Obsidian Community](https://obsidian.md/community) — official community hub
- [Custom CSS forum tag](https://forum.obsidian.md/tag/custom-css/l/top) —
  community snippet discussions and experiments
- [Share & showcase forum](https://forum.obsidian.md/c/share-showcase/9) —
  theme and vault showcase announcements
- [Community themes registry](https://github.com/obsidianmd/obsidian-releases/blob/master/community-css-themes.json) —
  official index of community themes

## How the Garden uses these resources

- The local theme library in [`.obsidian/themes/`](../.obsidian/themes/README.md)
  vendors a small set of community themes as plain runtime assets.
- The local snippet library in
  [`.obsidian/snippets/`](../.obsidian/snippets/README.md) vendors a curated set
  of Awesome Obsidian snippets for quick experimentation.
- The vault keeps upstream licensing and attribution while avoiding nested Git
  repositories.
