# Shared Workstation Configuration

Cross-platform user-level configuration that can be linked or copied into a
home directory on macOS, Linux, containers, or remote workstations.

## Layout

- `ack/` - ack search configuration.
- `alacritty/` - terminal emulator configuration.
- `aria2/` - aria2 downloader configuration.
- `asdf/` - asdf runtime manager configuration.
- `curl/` - curl client defaults.
- `git/` - user-level Git configuration and global ignore rules.
- `npm/` - user-level npm and package-manager defaults.
- `python/` - Python REPL startup configuration.
- `readline/` - readline/inputrc configuration.
- `shell/` - generic shell login/logout files.
- `wget/` - wget client defaults.
